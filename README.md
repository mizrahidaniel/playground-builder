## Usage

1. Open `index.html` in your browser
2. Drag components from the left palette onto the canvas
3. Click components to edit their properties in the right panel
4. Click "Export HTML" to copy standalone HTML to clipboard

## Components

- **Heading** - H1-H6 headings with configurable level
- **Text** - Paragraph text with editable content
- **Button** - Interactive button with custom label
- **Input** - Text input with placeholder
- **Code Block** - Syntax-highlighted code display
- **Container** - Layout container for grouping

## Export

The exported HTML includes:
- All components with their current values
- Minimal inline CSS for styling
- Standalone file ready to use
